# Discord OOP (discord-oop)

An opinionated TypeScript wrapper built on top of discord.js, adding custom types, enums, and utility functions for common patterns.

## Installation

```bash
# From your git repo
pnpm add git+https://github.com/yourusername/discord-oop.git
```

## Usage

```typescript
import { utilities, enums } from 'discord-oop';

// Your bot code here
```

## Development

```bash
pnpm run build      # Compile TypeScript
pnpm run dev        # Watch mode
```

## Requirements

- Node.js >= 24.14.0
- discord.js ^14.0.0 (peer dependency)
